# Calculator

A modern, interactive web-based calculator with a sleek design and theme switching capabilities.

## Features

- **Basic Operations**: Perform addition (+), subtraction (-), multiplication (×), and division (/) operations
- **Delete Function**: Remove the last digit from your input with the DEL button
- **Reset Function**: Clear all entries and start fresh with the RESET button
- **Decimal Support**: Work with decimal numbers for precise calculations
- **Theme Switching**: Toggle between two distinct visual themes for comfortable use in different lighting conditions
- **Responsive Design**: Works seamlessly on various screen sizes

## Project Structure

```
calculator/

```

## How to Use

1. **Open the Calculator**: Open `index.html` in your web browser
2. **Enter Numbers**: Click on number buttons (0-9) to input values
3. **Perform Operations**: Click an operation button (+, -, ×, ÷) to select the operation
4. **Calculate Result**: Press the `=` button to compute the result
5. **Clear Input**: Use `DEL` to remove the last digit or `RESET` to clear everything
6. **Switch Theme**: Use the THEME toggle to switch between dark and light themes

## Supported Operations

- **Addition** (+)
- **Subtraction** (-)
- **Multiplication** (×)
- **Division** (/)
- **Decimal Numbers** (.)

## Technical Details

- **HTML5**: Semantic markup with proper accessibility attributes
- **Vanilla JavaScript**: Pure JavaScript with no external dependencies
- **CSS3**: Custom properties (CSS variables) for easy theme management
- **Font**: League Spartan font from Google Fonts
